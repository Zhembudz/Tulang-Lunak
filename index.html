Tentang Saya

Saya adalah mahasiswa semester 3 Program Vokasi, Prodi Teknologi Rekayasa Perangkat Lunak, Universitas Negeri Gorontalo.

Pengalaman Organisasi saya pernah menjadi ketua bidang 2 Ekonomi Kreatif Himpunan Mahasiswa Jurusan Teknologi Rekayasa Perangkat Lunak Periode 2022, Sekarang menjadi anggota divisi 5 Senat Mahasiswa Program Vokasi.

Tujuan saya masuk jurusan Teknologi Rekayasa Perangkat Lunak karena saya ingin mempelajari tentang sofware developer dan game developer dan juga menurut saya lapangan pekerjaan untuk jurusan ini sangat luas.
